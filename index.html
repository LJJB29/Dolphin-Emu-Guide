The Full Dolphin Emulator Guide

An Easy-to-Follow Guide for Dolphin Emulator and How to (Legally) Use It

(This IS NOT a Guide for Android)

How Do I Install the Emulator?

Follow these steps to install Dolphin Emulator:

  1. Visit Dolphin Emulator's Official Website and click the "Download Dolphin" button for your operating system.
  2. Download the latest version for your operating system and architecture. Tip: You might need to install the 64-bit Visual C++ Redistributable for Visual Studio 2022 for Windows users.
  3. Once the download is complete, locate the zip file in your File Explorer (Windows) or Finder (Mac) and extract it.

Congratulations!

You’ve successfully installed Dolphin Emulator on your PC!

How Do I Add My Legally Aquired Games Into the Emulator?

Note: Only use legally acquired games to avoid any legal issues. Piracy is not recommended!

There are 2 legal ways to do this:

Option 1: Using an External Drive

  1. Purchase an External Drive for £10 on Amazon.
  2. Insert the disk into the external drive.
  3. Extract the game files from the disk. (Note: This step may not work for every game.)

Option 2: Using a Homebrewed Wii and USB Stick

  1. Install USB Loader GX to your homebrewed Wii via the Homebrew Channel.
  2. Insert the disk into your console.
  3. Open USB Loader GX and wait for the pop-up.
  4. Select Extract Disk to copy the game onto your USB.
  5. Plug the USB into your computer.
  6. Move the games to your C:\ Drive.
  7. You can also extract WADs (Channels) this way!

How to Add Games into Dolphin Emulator

  1. Open Dolphin Emulator.
  2. Click on the text that says No Games Found on the main menu.
  3. Select the directory where your games are stored.
  4. Done! Your games are now available in Dolphin Emulator!

How Do I Get the Wii System on Dolphin Emulator?

  1. Click the Tools button at the top of the screen.
  2. Select Update Wii System Menu.
  3. Choose your region (e.g., USA).
  4. Wait for the update to finish.
  5. Once completed, press Load Wii System Menu [Version].

Optional: Homebrewing the Virtual Wii

  1. Download the latest Homebrew Channel Wad from releases.
  2. Place it in your Games folder.
  3. Right-click the file in Dolphin's main menu and select Install to the NAND.
  4. This method also works for installing other WAD files!

How to Use an Actual Wii Remote with Dolphin

  1. Go to Controller Settings and select Real Wii Remote under Wii Remote #1.
  2. Turn Continuous Scanning to ON.
  3. On the Wii Remote, open the battery cover and press the little red sync button.
  4. Wait for the Wii Remote to vibrate, indicating it's connected.
  5. Buy the Dolphin Sensor Bar from Amazon for a better experience!

How to Use an Actual GameCube Controller

  1. Buy the GameCube Controller Adapter for PC from Amazon.
  2. Plug your controller into the adapter and set it as GameCube Controller #1 in Dolphin's settings.
